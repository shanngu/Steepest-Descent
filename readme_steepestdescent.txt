To run the file, you must have JavaSE-1.7 JRE configured.
All files should be included for use in the SteepestDescent folder.
To run the applet, run SteepestDescent.java, found in the src folder of the SteepestDescent folder.
You can use an IDE to run the applet (e.g. Eclipse).
Alternatively, you can also run the applet from the command line.
	- From the command line, go to the folder containing the SteepestDescent.java class (SteepestDescent/src)
	- Run 'javac *.java' to compile all the java files
	- Run 'java SteepestDescent' to run the applet
